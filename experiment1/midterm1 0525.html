Introduction

The Bachelor of Technology in Computer Science is designed to provide students with strong foundations
in computer science, software engineering, and emerging technologies.

  4. Programming in C
  5. Engineering Drawing
  6. Web Designing Workshop-1
  7. Essential of AI

Programming Languages Covered

  • C
  • C++
  • Java
  • Python
  • JavaScipt

KeyTerms

AI:- Artificial Intelligence, the study of intelligent machines.
ML:- Machine Learning, a subset of AI focused on learning from data.
DBMS:- Database Management System, software to handle databases.
OS:- Operating System, software that manages computer hardware and software resources.
